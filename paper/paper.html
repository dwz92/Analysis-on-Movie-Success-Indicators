3.1 Model set-up

Define \(y_i\) as the number of seconds that the plane remained aloft. Then \(\beta_i\) is the wing width and \(\gamma_i\) is the wing length, both measured in millimeters.

\[\begin{align} y_i|\mu_i, \sigma &\sim \mbox{Normal}(\mu_i, \sigma) \\ \mu_i &= \alpha + \beta_i + \gamma_i\\ \alpha &\sim \mbox{Normal}(0, 2.5) \\ \beta &\sim \mbox{Normal}(0, 2.5) \\ \gamma &\sim \mbox{Normal}(0, 2.5) \\ \sigma &\sim \mbox{Exponential}(1) \end{align}\]

We run the model in R (R Core Team 2023) using the rstanarm package of Goodrich et al. (2022). We use the default priors from rstanarm.
